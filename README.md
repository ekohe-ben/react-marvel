# React-Marvel

Showing comics using ReactJS & Marvel API

## Setup

1. `git clone https://github.com/navyboys/react-marvel.git`
1. `cd react-marvel`
1. `npm install -g http-server`
1. `http-server`
1. Access http://locahost:8080 in browser
